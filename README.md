---

## 🧠 How It Works

1. User taps the **Emergency Button**
2. Camera & mic recording starts instantly
3. User location is captured
4. Nearest volunteers are calculated
5. Volunteers move toward the user on **real routes**
6. ETA is calculated dynamically
7. Recording can only be stopped with a **secret password**
8. Video evidence is stored securely in browser storage

## 📁 Project Structure

# 🚨 SafeCircle — Community Safety Web App

SafeCircle is a **placement-oriented web project** focused on real-time safety assistance.  
With a single emergency tap, the system **starts live recording**, **shares location**, and **alerts nearby volunteers** with animated routing and ETA.
